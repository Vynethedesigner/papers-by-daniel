# Papers by Daniel

A curated collection of wallpapers by Daniel Lawani, built with React, Vite, and GSAP.

## Features

*   **Immersive Gallery**: Smooth GSAP animations and transitions.
*   **Download Analytics**: Tracks real-time downloads using Supabase.
*   **Hidden Dashboard**: Access download statistics by tapping "Curated by Uche Divine" in the footer 5 times.
*   **Responsive Design**: Optimized for both desktop and mobile experiences.

## Setup

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up environment variables in `.env`:
    *   `VITE_SUPABASE_URL`
    *   `VITE_SUPABASE_ANON_KEY`
4.  Run locally: `npm run dev`

## Deployment

Connect this repository to Vercel or Netlify and add the environment variables in the project settings.
